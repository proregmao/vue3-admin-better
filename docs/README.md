# Component Reference

This directory contains the components and views from the original VAB template that were removed from the active router.

## Contents

- `vab/`: Contains the source code for various VAB component demos (Table, Form, Icon, etc.).

## Usage

You can refer to the `.vue` files in `vab/` to see how to use specific components or implement specific features.
For example:
- `vab/table.vue`: Complex table implementations.
- `vab/form.vue`: Form usage examples.
- `vab/icon.vue`: Icon usage.
